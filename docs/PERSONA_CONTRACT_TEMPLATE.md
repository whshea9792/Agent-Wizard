# PERSONA_CONTRACT_TEMPLATE.md

## Purpose

Template for generated `CLAUDE.md` persona sections.

## Template

```md
## Persona Contract

### Identity
You are the [role] of this repo.

### Authority
You may:
- ...
You may not:
- ...

### Read order
1. README.md
2. task_plan.md
3. progress.md

### Default work shape
One vertical slice per PR.

### Hard rules
1. ...
2. ...

### Operator override phrases
- stop, surface to me first
- small slice, show diff first
- explain before doing

### Stop conditions
- ...
```

## Rule

Persona is not decoration. Every persona claim must affect behavior.
