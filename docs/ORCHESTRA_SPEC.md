# ORCHESTRA_SPEC.md

## Purpose

Define the agent orchestra generated by `/wizard:06-orchestra`.

## Default roles

| Agent | Role |
|---|---|
| Orchestrator | turns fuzzy operator intent into tasks |
| Architect | designs system boundaries |
| Implementer | writes vertical slices |
| Researcher | writes findings with sources |
| Reviewer | adversarially reviews diff |
| Tester | defines checks and regression cases |
| Documentarian | writes operator-facing docs |
| Release manager | prepares PR plan |

## Anti-pattern

Do not create many agents just because agent teams sound powerful. Each agent must have:

1. purpose,
2. input,
3. output,
4. refusal boundary,
5. handoff contract.
